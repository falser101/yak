# 骑行装备模块 PRD

## 1. 概述

### 1.1 产品背景
为骑行爱好者提供专业自行车租赁服务，支持日租、小时租、团队租等多种租赁方式，涵盖高原骑行所需的各类专业装备。

### 1.2 目标用户
- 前往青海湖、西藏等地的高原骑行爱好者
- 团队骑行活动组织方
- 短期体验用户

### 1.3 核心价值
- 丰富的车型选择（山地车、公路车、折叠车等）
- 灵活的租赁方案
- 透明的租金与押金规则
- 便捷的在线预订

---

## 2. 功能架构

### 2.1 用户端功能（微信小程序）

#### 2.1.1 租车首页
- 车型分类筛选：山地车、公路车、折叠车、电动车
- 租赁类型筛选：日租、小时租、团队租、里程租
- 车型卡片展示：车型图、名称、分类标签、日租价格

#### 2.1.2 车型详情
- 车型封面图与基本信息
- 规格参数：车架尺寸、变速器、刹车类型、轮径、适合身高
- 租车说明：取车还车规则、押金规则
- "加入我的车辆"收藏功能

#### 2.1.3 品牌目录
- 品牌列表：TREK、GIANT、Specialized、CANYON、MERIDA、Bianchi 等
- 品牌 LOGO 与简介

#### 2.1.4 系列目录
- 按品牌查看车系列表
- 年份筛选 Tab
- 系列卡片：系列名、代表车型图片

#### 2.1.5 租车预订
- 租车日期选择
- 租车套餐选择：日租套餐、小时套餐、团队套餐
- 数量选择
- 费用明细：租金 + 押金
- 租车须知确认
- 提交订单

#### 2.1.6 我的订单
- 租车订单列表
- 状态筛选：全部/待支付/已支付/已完成/已取消
- 订单详情查看

### 2.2 管理端功能

#### 2.2.1 车辆管理（Admin）
- 车辆列表：筛选（分类/状态）、搜索、分页
- 车辆 CRUD：名称、分类、封面图、规格参数、价格设置、押金、租车说明
- 分类管理：添加/编辑/删除车型分类

#### 2.2.2 订单管理（Admin）
- 订单列表：筛选（状态）、搜索、分页
- 订单详情查看
- 状态变更操作：确认支付、取消订单
- 导出订单数据

## 4. 页面原型说明

### 4.1 小程序 租车首页
- 顶部：平台 Logo + 筛选入口

### 4.2 小程序 车型详情页
- 车型封面大图

- 规格参数卡片区
- 租车说明区
- 底部操作栏："加入我的车辆" + "立即租车"

### 4.3 小程序 品牌目录页
- 品牌 LOGO 网格
- 品牌名称与简介

### 4.4 小程序 系列目录页
- 品牌信息区
- 年份 Tab 筛选
- 系列卡片列表

### 4.5 小程序 租车预订页
- 车型信息卡片
- 日期选择器
- 套餐选择（单选组）

- 左侧导航栏

## 5. 业务流程

### 5.1 租车预订流程
```
用户浏览车型 → 选择车型 → 选择租车日期 → 选择套餐 → 选择数量 →
确认费用明细 → 阅读租车须知 → 提交订单 → 支付 → 支付成功 → 生成订单
```

### 5.2 车辆管理流程
```
管理员添加车辆 → 填写车辆信息 → 设置价格套餐 → 上架车辆 →
用户预订 → 管理员审核订单 → 确认/取消订单
```

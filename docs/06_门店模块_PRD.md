# 门店模块 PRD

## 1. 概述

### 1.1 产品背景
为骑闯天路线下服务门店提供线上展示窗口，方便用户查找门店位置、了解服务内容、联系门店。

### 1.2 目标用户
- 需要线下服务（取车/还车/维修/咨询）的用户
- 门店工作人员

### 1.3 核心价值
- 门店位置与服务展示
- 在线联系门店
- 线下服务预约（预留）

---

## 2. 功能架构

### 2.1 用户端功能（微信小程序）

#### 2.1.1 门店列表（入口）
- 门店列表入口（我的页面 → 门店查询）
- 门店卡片展示：名称、地址、距离

#### 2.1.2 门店详情
- 门店封面图
- 基本信息：名称、电话、地址、营业时间
- 服务项目展示
- 地图位置（占位）
- 一键拨号按钮
- 一键导航按钮（预留）

### 2.2 管理端功能

#### 2.2.1 门店管理（Admin Vue）
- 门店列表：名称、地址、电话、营业时间、状态
- 新建门店
- 编辑门店
- 删除门店
- 启用/禁用门店

---

## 3. 数据模型

### 3.1 Store（门店）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 门店ID |
| name | string | 门店名称 |
| phone | string | 联系电话 |
| address | string | 详细地址 |
| city | string | 所在城市 |
| latitude | number | 纬度 |
| longitude | number | 经度 |
| hours | string | 营业时间 |
| services | string[] | 服务项目列表 |
| cover | string | 门店封面图 |
| status | enum | 营业中/歇业/装修中 |

---

## 4. 页面原型说明

### 4.1 小程序 门店详情页
- 门店封面大图
- 信息卡片：名称、电话、地址、营业时间
- 服务项目标签区
- 地图占位区
- 底部操作栏：拨打电话、导航到门店

### 4.2 管理端门店管理页
- 门店卡片网格或表格展示
- 新建/编辑门店弹窗表单：
  - 门店名称
  - 联系电话
  - 所在城市
  - 详细地址
  - 营业时间
  - 服务项目（多选）
  - 门店封面图上传

---

## 5. 业务流程

### 5.1 门店服务流程
```
用户浏览门店信息 → 查看门店位置和服务 → 电话联系门店 →
到店取车/还车/享受服务 → 离店
```

---

## 6. 非功能性需求

### 6.1 地图
- 门店位置需支持地图插件展示（预留）
- 支持第三方地图应用调起

### 6.2 体验
- 电话号码支持一键拨打
- 门店信息实时更新
- 门店状态（营业/歇业）实时显示

---

## 7. 依赖关系

- 依赖用户模块（定位服务，预留）
- 未来可能与租车模块联动（取车门店/还车门店）

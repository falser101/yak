# 管理后台模块 PRD

## 1. 概述

### 1.1 产品背景
骑闯天路骑行平台管理后台，为平台运营人员提供统一的后台管理入口，支持活动管理、用户管理、订单管理、门店管理等核心运营功能。

### 1.2 目标用户
- 平台运营管理员
- 活动策划人员
- 客服人员

### 1.3 核心价值
- 统一管理入口
- 高效运营工具
- 数据可视化展示

---

## 2. 功能架构

### 2.1 Admin Vue SPA（主后台）

#### 2.1.1 登录页面
- 用户名 + 密码登录
- 记住登录状态

#### 2.1.2 控制台（Dashboard）
- 核心数据统计卡片：
  - 正在进行活动数
  - 累计报名人数
  - 注册用户数
  - 总收入
- 待审核报名列表
- 最近活动列表

#### 2.1.3 活动管理
- 活动列表与 CRUD
- 活动分类管理
- 报名审核

#### 2.1.4 报名管理
- 报名记录列表
- 按活动筛选
- 导出 Excel

#### 2.1.5 门店管理
- 门店列表与 CRUD
- 服务项目管理

#### 2.1.6 用户管理
- 用户列表
- 会员等级管理
- 用户状态管理

### 2.2 Admin Static（轻量后台）

#### 2.2.1 登录页
- 用户名 + 密码 + 验证码登录
- 记住密码功能

#### 2.2.2 活动管理
- 活动 CRUD
- 分类/状态筛选

#### 2.2.3 报名管理
- 报名记录列表
- 状态管理

#### 2.2.4 车辆管理
- 车辆 CRUD
- 分类管理

#### 2.2.5 俱乐部管理
- 俱乐部 CRUD
- 成员管理

#### 2.2.6 订单管理
- 订单列表
- 状态管理
- 支付确认

#### 2.2.7 用户管理
- 用户列表
- 状态切换

---

## 3. 权限设计

### 3.1 角色划分
| 角色 | 权限范围 |
|------|----------|
| 超级管理员 | 全部功能 |
| 运营管理员 | 活动、报名、用户、订单 |
| 客服 | 报名管理、用户查看 |
| 财务 | 订单、支付相关 |

### 3.2 功能权限
- 增删改查权限分离
- 敏感操作需二次确认
- 操作日志完整记录

---

## 4. 页面原型说明

### 4.1 Admin Vue 布局
- 左侧 Sidebar 导航（可折叠）
- 顶部 Header：用户头像、下拉菜单（个人中心、退出登录）
- 主体内容区

### 4.2 Dashboard 页面
- 统计卡片行：4个指标卡片
- 待审核表格
- 最近活动表格

### 4.3 Admin Static 布局
- 左侧 Sidebar 导航（可折叠）
- 面包屑导航
- 顶部用户信息栏
- 主体内容区

---

## 5. 技术架构

### 5.1 Admin Vue
- Vue 3 + TypeScript
- Vite 构建
- Element Plus UI
- Pinia 状态管理
- Axios 请求
- Vue Router 路由

### 5.2 Admin Static
- 原生 HTML + CSS + JavaScript
- 无需构建，直接部署
- localStorage 存储模拟数据

---

## 6. 安全设计

### 6.1 认证
- 管理员账号密码登录
- Session/Cookie 会话管理
- 登录失败次数限制

### 6.2 授权
- 路由级别权限控制
- API 请求 Token 验证

### 6.3 审计
- 管理员操作日志
- 登录日志记录

---

## 7. 非功能性需求

### 7.1 性能
- 后台页面首屏加载 < 2s
- 数据列表分页加载
- 操作响应 < 1s

### 7.2 体验
- 响应式布局（支持移动端查看）
- 操作反馈（Toast 提示）
- 表单验证友好提示

### 7.3 兼容性
- Chrome/Firefox/Safari/Edge 最新两个版本
- 移动端 Safari、微信内置浏览器

---

## 8. 依赖关系

- 依赖后端 API 服务
- 依赖用户认证服务
- 被所有业务模块管理功能依赖

# 活动模块 PRD

## 1. 概述

### 1.1 产品背景
骑闯天路骑行平台活动模块，为用户提供高原骑行赛事、公益活动、亲子骑行、俱乐部活动等多元化活动信息与报名服务。

### 1.2 目标用户
- C端用户：骑行爱好者、户外运动参与者
- B端：活动组织方、俱乐部管理员

### 1.3 核心价值
- 活动信息聚合与展示
- 在线报名与费用缴纳
- 活动管理与审核

---

## 2. 功能架构

### 2.1 用户端功能（微信小程序）

#### 2.1.1 活动列表（首页）
- Banner 轮播图展示精选活动
- 热门/新上线/推荐活动标签筛选
- 活动卡片展示：封面图、名称、日期、地点、费用、报名人数/容量
- Tab 分类：赛事、公益、亲子、俱乐部

#### 2.1.2 活动详情
- 活动封面与基本信息（名称、日期、地点、费用）
- Tab 页切换：
  - **规程**：详细规则、注意事项
  - **路线轨迹**：骑行路线信息
  - **奖项**：奖品设置
  - **往期回顾**：历史活动回顾
- 报名入口按钮

#### 2.1.3 活动报名
- 个人信息填写：姓名、手机号、身份证号
- 紧急联系人信息：姓名、手机号
- 附加选项（如有）：装备租赁、保险等
- 健康声明确认
- 提交报名

#### 2.1.4 我的报名
- 报名记录列表
- 状态筛选：全部/待支付/已支付/已完成/已取消
- 报名详情查看

### 2.2 管理端功能

#### 2.2.1 活动管理（Admin）
- 活动列表：筛选（分类/状态）、搜索、分页
- 创建活动：名称、分类、封面图、日期范围、地点、费用、容量、规则、路线、奖项
- 编辑活动
- 删除活动
- 查看报名列表
- 活动状态管理：草稿/报名中/进行中/已结束

## 4. 页面原型说明

### 4.1 小程序 活动列表页（首页）
- 顶部：平台 Logo + 搜索入口
- 中部：Banner 轮播（自动播放，3秒切换）
- 分类 Tab：赛事 | 公益 | 亲子 | 俱乐部
- 活动卡片列表（瀑布流或单列）

### 4.2 小程序 活动详情页
- 封面图（16:9 比例）
- 基本信息区：名称、日期、地点
- Tab 切换区
- 底部：报名按钮（固定定位）

### 4.3 小程序 报名表单页
- 步骤条：个人信息 → 确认提交
- 表单项：姓名、手机号、身份证号、紧急联系人、紧急联系电话
- 附加选项（复选框）
- 健康声明（需勾选）
- 提交按钮

### 4.4 管理端活动列表页
- 左侧导航栏（可折叠）
- 顶部：筛选条件 + 搜索 + 新建按钮
- 表格：封面图、名称、分类、日期、费用、报名/容量、状态、操作

## 5. 业务流程

### 5.1 活动报名流程
```
用户浏览活动列表 → 进入活动详情 → 点击报名 → 填写报名信息 → 确认并支付 →
支付成功 → 生成报名记录 → 发送通知 → 完成报名
```

### 5.2 活动管理流程
```
管理员创建活动 → 设置活动信息 → 发布活动 → 用户报名 → 审核报名 →
管理报名名单 → 活动进行 → 活动结束
```

## 6. 非功能性需求

### 6.1 性能
- 活动列表页首屏加载 < 2s
- 活动详情页加载 < 1.5s
- 支持 1000+ 并发报名

### 6.2 安全
- 用户敏感信息加密存储
- 报名表单防重复提交
- 管理员操作日志记录

### 6.3 体验
- 遵循微信小程序设计规范
- 支持微信分享功能
- 图片懒加载
- 网络异常时友好提示

---

## 7. 依赖关系

- 依赖用户模块（微信登录、用户信息）
- 依赖支付模块（微信支付）
- 依赖订单模块（报名订单生成）
